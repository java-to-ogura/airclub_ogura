# airclub_ogura

This repository provides a simple command line Todo application written in Java.

## Usage

Compile the program:

```
javac Todo.java
```

Add a task:

```
java Todo add "Buy milk"
```

List tasks:

```
java Todo list
```

Mark a task as done (by number):

```
java Todo done 1
```

Delete a task:

```
java Todo delete 1
```
